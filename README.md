# Sales-Profitability-Analysis
A SQL and Python project analyzing retail profit margins for business decision-making.
Goal: Identify which product categories drive the highest profit to optimize marketing spend.
Tools Used: > * Python (Pandas): For data structuring.
SQL (SQLite): To query and aggregate profit margins.
Matplotlib: To visualize insights for stakeholders.
Key Insight: My analysis revealed that while 'Tech' has higher volume, 'Office' supplies have a 15% higher profit margin per unit.
